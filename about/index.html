About Discussion

A better experience
on the computer you already own.
Trenta OS is a free & open operating system that aims to stand in the place of Mac OS X and Windows.
It is based on Ubuntu Linux so it runs most of the applications you already love.
The "Rainier" Look.
A first for Linux. With the Rainier UI and Icon set, we try to achieve a premium feel for our open-source operating system.
We often ask the community for their input to make it the best it can be.
Creativity & Media.
Trenta OS with Rainier UI tries to give everyone who is developing for web, graphics and video a great environment to do so in.
With PlayOnlinux you are able to install Adobe Photoshop, or you can try our modified version of GIMP.
Windows Applications.
With PlayOnLinux, an easy-to-use interface for WINE, you are able to install a lot of your existing Windows applications in the comfort of your Trenta OS installation.
Gaming, your way.
Easily install Steam directly from their website to open the door to your machine's gaming potential.
If you prefer something more casual, you can download a game like Minecraft instead.
